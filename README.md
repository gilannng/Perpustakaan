# 📚 Digital Library Web Application

Sebuah aplikasi web manajemen perpustakaan interaktif yang dirancang untuk memudahkan proses peminjaman, pengembalian, dan pencarian buku secara digital. Proyek ini dibangun dengan fokus pada antarmuka pengguna (UI) yang bersih, responsif, dan pengalaman pengguna (UX) yang mulus.

## ✨ Fitur Utama (Key Features)

* 🔍 **Pencarian Cerdas (Smart Search):** Mencari buku berdasarkan judul, penulis, atau kategori secara *real-time*.
* 🛒 **Sistem Peminjaman & Pengembalian:** Simulasi logika untuk meminjam buku dan mengembalikannya ke dalam sistem stok.
* 📖 **Katalog Interaktif:** Menampilkan detail buku lengkap dengan sinopsis singkat dan status ketersediaan.
* 📱 **Desain Responsif:** Tampilan yang beradaptasi dengan sempurna baik di layar *desktop*, *tablet*, maupun *smartphone*.
* ⚡ **Manipulasi DOM Dinamis:** Interaksi UI yang cepat tanpa perlu memuat ulang halaman (*page reload*).

---

## 🛠️ Teknologi yang Digunakan (Tech Stack)

Proyek ini sepenuhnya dibangun menggunakan teknologi *frontend* murni tanpa mengandalkan *framework* berat, guna menunjukkan pemahaman inti mengenai pengembangan web:
* **Struktur:** HTML5 Semantik.
* **Desain & Animasi:** CSS3 (Flexbox/Grid, Animasi kustom, Media Queries).
* **Logika & Interaktivitas:** Vanilla JavaScript (ES6+).
* **Penyimpanan Lokal:** *[Opsional: Hapus baris ini jika tidak memakai LocalStorage]* Menggunakan `localStorage` bawaan peramban untuk menyimpan data buku dan riwayat peminjaman sementara.

## 🚀 Cara Menjalankan Proyek (Local Setup)

Karena proyek ini berbasis *client-side* murni, menjalankannya sangatlah mudah:

1.  **Kloning Repositori:**
    ```bash
    git clone [https://github.com/gilannng/nama-repo-perpustakaanmu.git](https://github.com/gilannng/nama-repo-perpustakaanmu.git)
    ```
2.  **Buka Direktori:**
    ```bash
    cd nama-repo-perpustakaanmu
    ```
3.  **Jalankan Aplikasi:**
    * Cukup klik ganda pada file `index.html` untuk membukanya di peramban web bawaan.
    * *Atau*, untuk pengalaman pengembangan yang lebih baik, buka folder proyek di **VS Code** dan gunakan ekstensi **Live Server**.
